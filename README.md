# حساباتي — Hesabati

<div dir="rtl">

نظام إدارة مالية ومحاسبية شامل مبني بتقنيات حديثة، يدعم إدارة أعمال متعددة من خلال واجهة مستخدم عربية سلسة.

## ✨ أبرز الميزات

- **إدارة أعمال متعددة** — إنشاء وإدارة أكثر من عمل تجاري ضمن حساب واحد
- **القيود المحاسبية والسندات** — تسجيل سندات القبض والصرف والقيود اليومية
- **الحسابات والأموال** — إدارة الحسابات المصرفية والصناديق والمحافظ
- **الشركاء والموردون** — متابعة حسابات العملاء والموردين والمستحقات
- **المستودعات والمشتريات** — إدارة المخزون وفواتير الشراء
- **الموارد البشرية** — إدارة الموظفين والرواتب والأقسام والمسميات الوظيفية
- **التقارير والملخصات** — تقارير مالية مفصّلة وتقارير متقدمة قابلة للتخصيص
- **أسعار الصرف** — متابعة العملات وعمليات الصرافة
- **الصلاحيات والأدوار** — نظام أدوار مرن لإدارة صلاحيات المستخدمين
- **لوحة تحكم تفاعلية** — إحصاءات ومخططات ثلاثية الأبعاد في الوقت الفعلي

## 🛠️ التقنيات المستخدمة

| الجزء | التقنية |
|-------|---------|
| الواجهة الأمامية | Angular 21 + TypeScript + Tailwind CSS |
| المخططات | Chart.js + Three.js |
| الباكند | Hono + Node.js + TypeScript |
| قاعدة البيانات | PostgreSQL + Drizzle ORM |
| المصادقة | JWT (JSON Web Tokens) |
| الوقت الفعلي | WebSocket |
| النشر | Docker + Nginx |

## 🚀 التشغيل السريع

### المتطلبات

- [Docker](https://www.docker.com/) و Docker Compose
- أو: Node.js 20+، pnpm، PostgreSQL 16

### باستخدام Docker (الطريقة الأسرع)

```bash
# 1. انسخ ملف الإعدادات وعدّل القيم
cp .env.docker .env

# 2. شغّل الحاويات
docker compose up -d

# 3. نفّذ الـ migrations والبيانات الأولية
docker compose exec backend pnpm run db:migrate
docker compose exec backend pnpm run db:seed
```

افتح المتصفح على: **http://localhost**

### تشغيل محلي (بدون Docker)

راجع ملف [SETUP-DB.md](./SETUP-DB.md) للخطوات التفصيلية.

```bash
# تثبيت الاعتماديات
pnpm install

# في نافذة طرفية — الباكند
cd backend && pnpm run dev

# في نافذة طرفية أخرى — الواجهة
cd frontend && pnpm start
```

افتح المتصفح على: **http://localhost:4200**

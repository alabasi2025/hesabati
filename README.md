# حساباتي - إدارة محطات الكهرباء والحسابات الشخصية

نظام شخصي مخصص لإدارة محطات الكهرباء (الدهمية، الصبالية وجمال، غليل، الساحل الغربي، معبر) والحسابات والمحافظ المتعددة.

> **ملاحظة:** هذا نظام شخصي 100% - مصمم خصيصاً لطبيعة عمل المالك وليس نظام محاسبي عام.

## التقنيات المستخدمة

| التقنية | الوصف |
|---|---|
| **Angular 21** | الواجهة الأمامية |
| **Hono** | خادم الواجهة الخلفية |
| **oRPC** | بروتوكول الاتصال بين الواجهتين |
| **PostgreSQL** | قاعدة البيانات |
| **TypeScript** | لغة البرمجة |
| **TailwindCSS** | تنسيق الواجهة |
| **Drizzle ORM** | إدارة قاعدة البيانات |

## هيكل المشروع

```
hesabati/
├── backend/          # الخادم (Hono + oRPC + PostgreSQL)
│   └── src/
│       ├── db/       # قاعدة البيانات (Schema + Seed)
│       ├── routes/   # المسارات (Auth + Dashboard)
│       └── middleware/
├── frontend/         # الواجهة (Angular 21 + TailwindCSS)
│   └── src/app/
│       ├── components/  # (Sidebar + Header)
│       ├── pages/       # (Login + Dashboard)
│       ├── services/    # (Auth + API)
│       └── guards/
├── docs/             # الوثائق والملفات المرجعية
└── README.md
```

## التشغيل

### الخادم (Backend)
```bash
cd backend
pnpm install
pnpm dev
```

### الواجهة (Frontend)
```bash
cd frontend
pnpm install
pnpm start
```

## الأقسام الرئيسية

| القسم | الوصف |
|---|---|
| **لوحة التحكم** | ملخص سريع لكل شيء |
| **المحطات** | الدهمية، الصبالية وجمال، غليل، الساحل الغربي، معبر |
| **الحسابات والمحافظ** | جوالي، ون كاش، جيب، حاسب، كريمي، الصرافين، الخزائن، العهد |
| **التحصيل والتوريد** | التحصيل اليومي من المحطات وتوريده |
| **سندات القبض والصرف** | سند قبض، سند صرف، تحويلات |
| **الصرافين** | النجم، الحوشبي، النهمي، ابن عامر |
| **الموظفين والرواتب** | قائمة الموظفين، الرواتب، السلف، البصمة |
| **المخزن** | المخزن الرئيسي + مخازن المحطات |
| **الموردين** | ديزل، زيوت، عدادات، مواد كهربائية |
| **الشركاء** | محمد المراني، عمر إسحاق، إبراهيم نجم الدين |
| **التصفيات** | تصفية مع المدراء، الصرافين، المحاسب، الموردين |
| **التقارير** | التقرير الشهري للشراكة، تقرير كل محطة، الاستهلاك |
| **الحسابات المعلقة** | حسابات تحتاج تصفية |
| **الإعدادات** | العملات، المستخدمين، التصنيفات، النسخ الاحتياطي |

## الميزات

- واجهة عربية بالكامل (RTL)
- تصميم احترافي مع تأثيرات 3D وجسيمات متحركة
- إدارة 5 محطات كهرباء بأنظمة فوترة مختلفة
- إدارة الحسابات والمحافظ الإلكترونية (28 حساب)
- إدارة الصناديق (تحصيل، سلف، عهدة، خزنة)
- سندات قبض وصرف وتحويلات
- تتبع التحصيل اليومي والتوريدات
- إدارة المخزون (رئيسي + فرعي)
- إدارة الموظفين والرواتب (25 موظف)
- تصفيات مع المدراء والصرافين والمحاسب
- دعم العملات المتعددة (ريال يمني، سعودي، دولار)
- تتبع الحسابات المعلقة
- تقارير شهرية للشراكة

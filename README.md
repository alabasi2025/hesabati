# حساباتي - نظام إدارة مالية شخصية شاملة

نظام مالي شخصي لإدارة محطات الكهرباء والحسابات والمحافظ المتعددة.

## التقنيات المستخدمة

| التقنية | الوصف |
|---|---|
| **Angular 21** | الواجهة الأمامية |
| **Hono** | خادم الواجهة الخلفية |
| **oRPC** | بروتوكول الاتصال بين الواجهتين |
| **PostgreSQL** | قاعدة البيانات |
| **TypeScript** | لغة البرمجة |
| **TailwindCSS** | تنسيق الواجهة |
| **Drizzle ORM** | إدارة قاعدة البيانات |

## هيكل المشروع

```
hesabati/
├── backend/          # الخادم (Hono + oRPC + PostgreSQL)
├── frontend/         # الواجهة (Angular 21 + TailwindCSS)
├── docs/             # الوثائق والملفات المرجعية
└── README.md
```

## التشغيل

### الخادم (Backend)
```bash
cd backend
pnpm install
pnpm dev
```

### الواجهة (Frontend)
```bash
cd frontend
pnpm install
pnpm start
```

## الميزات

- واجهة عربية بالكامل (RTL)
- إدارة 5 محطات كهرباء
- إدارة الحسابات والمحافظ الإلكترونية
- إدارة الصناديق والصرافين
- تتبع الحركات المالية (مدخولات / مخروجات)
- إدارة المخزون
- إدارة الموظفين والرواتب
- دعم العملات المتعددة (ريال يمني، سعودي، دولار)
- تصميم احترافي مع تأثيرات 3D

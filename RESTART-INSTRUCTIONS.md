## 🔧 خطوات إعادة التشغيل

### 1️⃣ أوقف جميع العمليات الحالية:

**أ. أوقف Backend:**
- ابحث عن terminal يشغل `npm run dev` للـ backend
- اضغط `Ctrl+C` لإيقافه

**ب. أوقف Frontend:**
- ابحث عن terminal يشغل `npm start` للـ frontend  
- اضغط `Ctrl+C` لإيقافه

---

### 2️⃣ أعد تشغيل Backend:

```bash
# في terminal جديد
cd d:\Hhhh\hesabati\backend
npm run dev
```

**انتظر حتى ترى:**
```
🚀 حساباتي API يعمل على المنفذ 3000
```

---

### 3️⃣ أعد تشغيل Frontend:

```bash
# في terminal آخر
cd d:\Hhhh\hesabati\frontend
npm start
```

**انتظر حتى ترى:**
```
** Angular Live Development Server is listening on localhost:4200
```

---

### 4️⃣ افتح المتصفح:

1. اذهب إلى: `http://localhost:4200`
2. سجل دخول
3. اختر العمل (المحطات)
4. افتح: `12. المعلقات والتصفيات → العهد`
5. اضغط **Ctrl+Shift+R** (إعادة تحميل كاملة مع مسح الـ cache)

## 🔍 إذا لا تزال فارغة

### افتح Console في المتصفح (F12) وتحقق من:

1. **هل هناك أخطاء؟**
   ```
   انظر في تبويب "Console" لأي رسائل حمراء
   ```

2. **هل الـ API request نجح؟**
   ```
   انظر في تبويب "Network"
   ابحث عن: /api/businesses/1/accounts
   تحقق من الـ Response
   ```

3. **ماذا يرجع الـ API؟**
   ```javascript
   // اكتب في Console:
   fetch('/api/businesses/1/accounts', {
     headers: {
       'Authorization': 'Bearer ' + localStorage.getItem('token')
     }
   }).then(r => r.json()).then(data => {
     console.log('عدد الحسابات:', data.length);
     console.log('حسابات العهد:', data.filter(a => a.account_type === 'custody'));
   });
   ```

أرسل لي النتائج وسأساعدك في حل المشكلة!

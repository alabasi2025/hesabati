# خطوات إصلاح عرض حسابات العهد

## ✅ التعديل تم في الكود

تم تحديث `frontend/src/app/pages/custody/custody.ts`:

```typescript
// السطر 68-71
const custodyAccs = (accountsData || []).filter((a: any) => 
  a.accountType === 'custody' || a.account_type === 'custody'
);
```

---

## 🔧 لإظهار الحسابات

### الخطوات:

#### 1. تأكد من تشغيل Backend:
```bash
# في terminal منفصل
cd d:\Hhhh\hesabati\backend
npm run dev
```

#### 2. أعد تحميل صفحة Frontend:
- افتح المتصفح
- اذهب لصفحة العهد: `/biz/1/custody`
- اضغط **Ctrl+Shift+R** (إعادة تحميل كاملة)

#### 3. افتح Console (F12) وتحقق من:
- هل هناك أخطاء في Console؟
- هل API request `/api/businesses/1/accounts` يعمل؟

## 🔍 إذا لا تزال فارغة

افتح Console في المتصفح وأرسل لي:
1. أي رسائل خطأ في Console
2. ناتج هذا الأمر في Console:
```javascript
// افتح صفحة العهد ثم اكتب في Console:
localStorage.getItem('token')
```

هذا سيساعدني أعرف إذا كانت المشكلة في التوثيق أو في الكود.

## 🎯 الميزات الموجودة

### 1️⃣ عرض الفواتير:
- ✅ جدول بجميع فواتير المشتريات
- ✅ فلترة حسب الحالة (الكل، مسودة، مؤكدة، مكتملة)
- ✅ بحث برقم الفاتورة أو اسم المورد
- ✅ عرض: رقم الفاتورة، المورد، التاريخ، الإجمالي، المدفوع، المتبقي

### 2️⃣ إنشاء فاتورة جديدة:
- ✅ اختيار المورد (مطلوب)
- ✅ اختيار المستودع
- ✅ اختيار العملة
- ✅ طريقة الدفع (نقداً / آجل / جزئي)
- ✅ تاريخ الفاتورة وتاريخ الاستحقاق
- ✅ المرجع الخارجي (رقم فاتورة المورد)

### 3️⃣ إضافة أصناف:
- ✅ اختيار الصنف من قائمة المخزون
- ✅ تحديد الكمية
- ✅ تحديد سعر الوحدة
- ✅ إضافة ضريبة وخصم لكل صنف
- ✅ حساب تلقائي للمجاميع

### 4️⃣ حالات الفاتورة:
```
draft (مسودة)
  ↓ [تأكيد الفاتورة]
confirmed (مؤكدة) → في INT-01 (حساب وسيط)
  ↓ [استلام البضاعة]
partial (جزئية) → استلام جزئي
  ↓ [استلام كامل]
completed (مكتملة) → تم النقل للمخزن
```

### 5️⃣ عرض التفاصيل:
- ✅ معلومات الفاتورة الكاملة
- ✅ قائمة الأصناف
- ✅ المبالغ والإجماليات

### 6️⃣ الإجراءات:
- ✅ تأكيد الفاتورة
- ✅ تعديل (للمسودات فقط)
- ✅ عرض التفاصيل
- ✅ حذف (للغير مكتملة)
- ✅ استلام البضاعة

## 🔄 الآلية التشغيلية

### سيناريو كامل:

```
1. إنشاء فاتورة مشتريات:
   - اختيار المورد: محمود الحجة (ديزل)
   - إضافة صنف: ديزل، 1000 لتر، 800 ر.ي/لتر
   - الحالة: draft

2. تأكيد الفاتورة:
   - الحالة: confirmed
   - تسجيل في الحساب الوسيط INT-01
   - في انتظار الاستلام

3. استلام البضاعة:
   - تحديد الكميات المستلمة
   - الحالة: partial أو completed
   - نقل للمخزن
   - تحديث الأرصدة

4. إدخال المخزن:
   - نقل من INT-01 (وسيط)
   - إضافة للمخزن المحدد
   - تحديث رصيد المخزن
```

## 🎨 الواجهة

### الألوان حسب الحالة:
- 🟦 **مؤكدة** (confirmed) - أزرق
- 🟨 **مسودة** (draft) - رمادي
- 🟩 **مكتملة** (completed) - أخضر
- 🟧 **جزئية** (partial) - برتقالي
- 🟥 **ملغاة** (cancelled) - أحمر

### المعلومات المعروضة:
- رقم الفاتورة (fullSequenceNumber)
- اسم المورد
- التاريخ
- الإجمالي / المدفوع / المتبقي
- الحالة
- الأصناف والكميات

## 🔧 API Endpoints

### Backend Routes:
```typescript
GET    /businesses/:bizId/purchase-invoices           // جلب الفواتير
GET    /businesses/:bizId/purchase-invoices/:id       // تفاصيل فاتورة
POST   /businesses/:bizId/purchase-invoices           // إنشاء فاتورة
PUT    /businesses/:bizId/purchase-invoices/:id       // تعديل فاتورة
POST   /businesses/:bizId/purchase-invoices/:id/confirm  // تأكيد
POST   /businesses/:bizId/purchase-invoices/:id/receive  // استلام
DELETE /businesses/:bizId/purchase-invoices/:id       // حذف
```

### الفلاتر المدعومة:
- `?status=draft` - المسودات فقط
- `?status=confirmed` - المؤكدة فقط
- `?supplierId=5` - فواتير مورد معين

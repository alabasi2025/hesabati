## 🔍 تفاصيل كل نوع

### 1. صندوق (Fund)
- **المفتاح:** `fund`
- **الكود:** `FND-01`, `FND-02`, `FND-03`...
- **الاستخدام:** إدارة الصناديق النقدية
- **متطلبات:** يتطلب محطة (Station)
- **التصنيفات:** fund_types (للتنظيم فقط)

### 2. بنك (Bank)
- **المفتاح:** `bank`
- **الكود:** `BNK-01`, `BNK-02`, `BNK-03`...
- **الاستخدام:** إدارة الحسابات البنكية
- **متطلبات:** يتطلب مزود الخدمة (Provider) ورقم الحساب
- **التصنيفات:** bank_types (للتنظيم فقط)

### 3. محفظة إلكترونية (E-Wallet)
- **المفتاح:** `e_wallet`
- **الكود:** `WLT-01`, `WLT-02`, `WLT-03`...
- **الاستخدام:** إدارة المحافظ الإلكترونية (فودافون كاش، باي بال، إلخ)
- **متطلبات:** يتطلب مزود الخدمة (Provider)
- **التصنيفات:** e_wallet_types (للتنظيم فقط)

### 4. صراف (Exchange)

- **الكود:** `EXC-01`, `EXC-02`, `EXC-03`...

## ✅ تطبيق الآلية

### جميع الأنواع تطبق الآلية الصحيحة:

1. ✅ **كل نوع له بادئة** محددة في `TYPE_PREFIXES`
2. ✅ **الترقيم تلقائي** باستخدام `generateLeafAccountCode`
3. ✅ **التصنيفات للتنظيم فقط** وليس لها علاقة بالترقيم
4. ✅ **الكود فريد** لكل عمل/منشأة

### كيف تعمل الآلية:

```typescript
// عند إنشاء حساب فرعي بنوع "صندوق"
const result = await generateLeafAccountCode(bizId, 'fund', db);
// result.code = "FND-01"
// result.sequenceNumber = 1

// عند إنشاء حساب فرعي بنوع "بنك"
const result = await generateLeafAccountCode(bizId, 'bank', db);
// result.code = "BNK-01"
// result.sequenceNumber = 1
```

## 🎯 الخلاصة

- **13 نوع فرعي** للحسابات في النظام
- **جميعها تطبق الآلية الصحيحة** للترقيم
- **كل نوع له بادئة** خاصة به (FND, BNK, WLT, إلخ)
- **التصنيفات للتنظيم فقط** وليس لها علاقة بالترقيم
- **الترقيم يعتمد على** `account_sub_natures` وليس التصنيفات

## 🎯 الآلية المطبقة

### 📊 الحسابات (Accounts)

#### 1. الحسابات الرئيسية (isLeafAccount = false)
```
الكود: ترقيم شجري للتنظيم فقط
أمثلة: 1, 1.1, 1.2, 2, 2.1.1
الهدف: تنظيم الشجرة المحاسبية
```

#### 2. الحسابات الفرعية (isLeafAccount = true)
```
الكود: حسب النوع الفرعي (account_sub_natures)
أمثلة: FND-01, BNK-01, WLT-01, WHS-01, SUP-01
الهدف: الحسابات المالية الفعلية
```

## 🔧 Scripts المتاحة للفحص

### 1. فحص الأكواد
```bash
cd backend
npm run db:check  # فحص تطابق schema
npx tsx scripts/check-numbering.ts  # فحص الأكواد المولدة
```

### 2. إعادة تهيئة قاعدة البيانات
```bash
cd backend
npm run db:fresh  # حذف كل شيء + push + seed (الأمر الموصى به)
```

أو خطوة بخطوة:
```bash
npm run db:reset  # حذف جميع الجداول
npm run db:push   # إنشاء الجداول
npm run db:seed   # تعبئة البيانات الأولية
```

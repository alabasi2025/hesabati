## 02 - بنية النظام

> **الوصف:** يحتوي على التوثيق التقني لبنية نظام حساباتي - من شاشات الواجهة إلى أنواع العمليات والأقسام والبيانات الافتراضية والتقنيات المستخدمة. هذا القسم هو المرجع الأساسي للمطور.

| الملف | الوصف |
|---|---|
| [screens_stations.md](./02-بنية-النظام/screens_stations.md) | **شاشات قسم المحطات** - توثيق كامل لكل شاشات عمل المحطات: التحصيل والتوريد، السندات، القيود، الحسابات، أنظمة الفوترة، وإدارة الصناديق والبنوك والصرافين والمحافظ |
| [operation_types.md](./02-بنية-النظام/operation_types.md) | **أنواع العمليات (القوالب)** - توثيق كل قوالب العمليات: قوالب التحصيل (3 محطات)، قوالب التوريد (6 قوالب)، قوالب السندات، وقوالب القيود |
| [sections_types.md](./02-بنية-النظام/sections_types.md) | **أنواع الأقسام** - توثيق الأقسام المتاحة: التحصيل والتوريد، السندات، القيود، الحسابات، أنواع العمليات، مع الأقسام المخطط لها مستقبلاً |
| [seed_data.md](./02-بنية-النظام/seed_data.md) | **البيانات الافتراضية** - المرجع الرئيسي لكل البيانات الأولية: الأعمال، المحطات، الموظفين، أنظمة الفوترة، الصناديق، الحسابات، أنواع العمليات |
| [tech_stack.md](./02-بنية-النظام/tech_stack.md) | **التقنيات المستخدمة** - Angular 21، Hono، oRPC، PostgreSQL، مبادئ التصميم والبناء |

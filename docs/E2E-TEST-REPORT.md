## 3. عمليات إدخال بيانات من المتصفح

### إضافة محطة جديدة

- فتح صفحة المحطات → النقر على «إضافة محطة»
- تعبئة: اسم المحطة «محطة اختبار E2E»، الرمز «ST-E2E-01»، الموقع «عنوان تجريبي»
- الحفظ والتحقق من ظهور المحطة في القائمة

### إضافة شريك جديد

- فتح صفحة الشركاء → النقر على «إضافة شريك»
- تعبئة: الاسم «شريك اختبار E2E»، نسبة الحصة 25، الدور «شريك مؤسس»
- الحفظ والتحقق من ظهور الشريك في القائمة

### فتح نماذج إضافة أخرى

- **الحسابات:** فتح صفحة الحسابات والنقر على «حساب جديد» والتحقق من ظهور النموذج
- **الصناديق:** فتح صفحة الصناديق والتحقق من وجود زر «صندوق جديد» أو «إضافة صندوق»

## الخلاصة

- **تسجيل الدخول واختيار العمل:** يعمل كما هو متوقع.
- **جميع المسارات المذكورة:** تُحمّل دون خطأ ظاهر في الواجهة.
- **إدخال البيانات:** تم تنفيذ إضافة محطة وإضافة شريك بنجاح؛ نماذج الحسابات والصناديق تفتح بشكل صحيح.

لإعادة تشغيل الاختبارات محلياً:

```bash
cd frontend && npx playwright test e2e/all-pages-and-operations.spec.ts
```

```

- الكود المعروض: PRT-01 ✅ (من جدول accounts)  
- الترقيم المعروض: 1 ✅
```

## 📊 الآلية الصحيحة

### البنية:

```
جدول employees/businessPartners:
  - معلومات الموظف/الشريك الأساسية
  - له كود داخلي بسيط (EMP01, PRT01)

جدول accounts:
  - الحساب المالي المرتبط
  - الكود الصحيح الموحد (EMP-01, PRT-01)
  - accountType = 'employee' أو 'partner'
  - linkedEmployeeId يشير للموظف
```

### العرض في الواجهة:

```typescript
// الكود الصحيح من الحساب المرتبط
emp.code        → EMP-01  (من accounts)
emp.sequenceNumber → 1    (من accounts)

partner.accountCode → PRT-01 (من accounts)
partner.accountSequence → 1  (من accounts)
```

## 🔧 التعديلات المنفذة

### 1. Backend - employees.routes.ts

**تحديث GET /businesses/:bizId/employees:**

```typescript
.select({
  // ... الحقول الأخرى
  // إضافة الكود من الحساب المرتبط
  code: accounts.code,
  sequenceNumber: accounts.sequenceNumber,
  accountId: accounts.id,
})
.leftJoin(accounts, and(
  eq(accounts.businessId, bizId),
  eq(accounts.accountType, 'employee'),
  eq(accounts.linkedEmployeeId, employees.id)
))
```

**النتيجة:**
- API ترجع `code: "EMP-01"` من accounts ✅
- API ترجع `sequenceNumber: 1` من accounts ✅

### 2. Backend - partners.routes.ts

**تحديث GET /businesses/:bizId/partners:**

```typescript
.select({
  // ... الحقول الأخرى من businessPartners
  // إضافة الأكواد من الحساب المرتبط
  accountCode: accounts.code,
  accountSequence: accounts.sequenceNumber,
})
.leftJoin(accounts, and(
  eq(accounts.businessId, bizId),
  eq(accounts.accountType, 'partner'),
  eq(accounts.id, businessPartners.accountId)
))
```

**النتيجة:**
- API ترجع `accountCode: "PRT-01"` من accounts ✅
- API ترجع `accountSequence: 1` من accounts ✅

## المرحلة 1: الحسابات والصلاحيات 🔄 (الحالية)

### 1.1 إنشاء الحسابات
- [ ] أنواع الحسابات: صندوق، بنك، محفظة، صراف، حساب محاسبي، حساب وسيط
- [ ] بيانات الحساب: الاسم، النوع، الرصيد الافتتاحي، العمل التابع له
- [ ] أيقونة "مبدأ العمل" + شرح الصفحة

### 1.2 صلاحيات الحسابات (4 صلاحيات لكل حساب)
- [ ] يقدر يسوي قبض (هو يسحب من غيره): نعم/لا
- [ ] يقدر يسوي صرف (هو يدفع لغيره): نعم/لا
- [ ] يقدر يستقبل عملية صرف إليه (غيره يدفع له): نعم/لا
- [ ] يقدر يتنفذ عليه عملية قبض (غيره يسحب منه): نعم/لا

### 1.3 صلاحيات القيود (لكل حساب)
- [ ] يقدر يكون دائن في قيد: نعم/لا + قائمة الحسابات المدينة المسموحة
- [ ] يقدر يكون مدين في قيد: نعم/لا + قائمة الحسابات الدائنة المسموحة

### 1.4 ربط الحسابات (المستقبلات)
- [ ] تحديد قائمة الحسابات المسموح التعامل معها لكل حساب
- [ ] لكل ربط: تحديد نوع العلاقة (صرف إليه، قبض منه، يستقبل صرف، يتنفذ عليه قبض)
- [ ] الفلترة التلقائية: لما تختار حساب تظهر فقط الحسابات المسموحة
